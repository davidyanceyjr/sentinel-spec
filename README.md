# Sentinel Specification Repository

This repository contains a scaffolded, sectioned version of the Sentinel canonical engineering specification.

## Structure

- `SPEC.md` is the entrypoint and index.
- `core/` contains the top-level canonical Sentinel specification.
- `subsystems/secobs/` contains the SecObs subsystem specification.
- `subsystems/journald-findings-engine/` contains the journald security findings engine material.
- `subsystems/journald-parser/` contains the journald parser specification.
- `reference/` contains rule catalogs and other reference material.
- `archive/raw-heading-split/` preserves the direct heading-split output.

The split section files were reorganized by heading/topic only. Their text was not rewritten.
