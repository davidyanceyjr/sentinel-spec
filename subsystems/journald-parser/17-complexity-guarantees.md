# 17. Complexity Guarantees

## Normative Requirements

- `[JPARSER-CPLX-001]` The parser MUST guarantee linear time complexity.
- `[JPARSER-CPLX-002]` The parser MUST guarantee single-pass parsing.
- `[JPARSER-CPLX-003]` The parser MUST guarantee bounded memory usage.
- `[JPARSER-CPLX-004]` Malformed inputs MUST NOT cause superlinear behavior.

The parser must guarantee:

-   linear time complexity
-   single-pass parsing
-   bounded memory

Malformed inputs must not cause superlinear behavior.
