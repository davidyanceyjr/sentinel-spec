# 18. Output Guarantees

## Normative Requirements

- `[JPARSER-OUT-001]` For each record, `record_index` MUST increase monotonically.
- `[JPARSER-OUT-002]` For each record, fields MUST be structurally valid.
- `[JPARSER-OUT-003]` For each record, warnings MUST describe anomalies.
- `[JPARSER-OUT-004]` Field order MUST NOT be relied upon.

For each record:

-   record_index increases monotonically
-   fields are structurally valid
-   warnings describe anomalies

Field order must not be relied upon.
