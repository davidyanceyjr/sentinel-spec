# 9. Testing Strategy by Stage

## Normative Requirements

- `[JFE-TEST-001]` Parser tests MUST validate byte decoding and raw-record emission.
- `[JFE-TEST-002]` Normalization tests MUST validate known-field extraction and typed conversion.
- `[JFE-TEST-003]` Classification tests MUST validate event-class assignment.
- `[JFE-TEST-004]` Rule tests MUST validate finding emission against fixture corpora.
- `[JFE-TEST-005]` Renderer tests MUST validate stable textual and JSON output.
- `[JFE-TEST-006]` End-to-end tests MUST validate complete pipeline behavior for representative datasets.
- `[JFE-TEST-007]` Each stage SHOULD be testable independently.
